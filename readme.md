Consulta de Zoneamento - Três Corações (MG)

Aplicação desenvolvida em Python com Streamlit para facilitar a consulta de zoneamento urbano e atividades permitidas (CNAEs) na cidade de Três Corações.

🚀 Como Executar Localmente

Instale as dependências:

pip install -r requirements.txt


Execute a aplicação:

streamlit run app.py


📂 Estrutura Necessária

Para que a aplicação funcione, os seguintes ficheiros devem estar na raiz do projeto:

app.py

requirements.txt

Estrutura-Detalhada-cnaes.xlsx

Cnaes-com-zoneamento.xlsx

brasão.png (opcional)
